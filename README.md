В файле .env нужно прописать значения для следующих переменных:
```ini
SECRET_KEY=<>

# данные для почты
EMAIL_HOST_USER=<>
DEFAULT_FROM_EMAIL=<>
EMAIL_HOST_PASSWORD=<>

# данные для oauth yandex
YANDEX_CLIENT_ID=<>
YANDEX_SECRET=<>

# данные для postgres
DB_HOST=<>
DB_USER=<>
DB_PORT=<>
DB_PASSWORD=<>
DB_NAME=<>

```
